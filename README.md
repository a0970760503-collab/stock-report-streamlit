# 股票報表系統

Streamlit app for uploading stock research PDFs, extracting stock code, target price, broker, and report date, then fetching current price from Yahoo Finance.

## Streamlit Cloud

Use this file as the app entry point:

```text
streamlit_app.py
```

Required files:

```text
streamlit_app.py
requirements.txt
```

## Local Run

```powershell
streamlit run streamlit_app.py
```
